# Password Strength Analyzer

## Description

This project is developed as part of the Thiranex Cyber Security Internship.

The Password Strength Analyzer evaluates the strength of user-entered passwords based on:

* Password Length
* Uppercase Letters
* Lowercase Letters
* Numbers
* Special Characters

The tool classifies passwords as:

* Weak
* Medium
* Strong

It also provides suggestions to improve password security and recommends stronger passwords.

## Features

* Check password length
* Check password complexity
* Identify weak passwords
* Suggest improvements
* Generate stronger password suggestions

## Technologies Used

* Python 3

## How to Run

1. Open the project in Python IDLE.
2. Run the file `password_strength_analyzer.py`.
3. Enter a password when prompted.
4. View the password strength and suggestions.
